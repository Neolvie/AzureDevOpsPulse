# AzureDevOpsPulse

Веб-приложение на Flask для отображения статистики разработчиков из локального TFS (Team Foundation Server / Azure DevOps Server). Подключается к TFS REST API, синхронизирует данные в SQLite и показывает дашборды через браузер.

## Быстрый старт

### 1. Создать `.env` из примера

Скопируйте файл `.env.example` в `.env` и заполните параметры:

```bash
cp .env.example .env
```

Содержимое `.env`:

```env
TFS_URL=https://ваш-tfs-сервер/   # базовый URL вашего TFS
PORT=14731                         # порт, на котором запустится приложение
LOG_LEVEL=INFO
```

### 2. Получить Personal Access Token (PAT) в TFS

PAT — это токен для доступа к TFS API вместо пароля. Чтобы его создать:

1. Войдите в веб-интерфейс вашего TFS.
2. Нажмите на иконку пользователя (правый верхний угол) → **Security** (Безопасность).
3. Выберите **Personal access tokens** → **Add**.
4. Задайте имя токена, срок действия и необходимые права (минимум: **Code – Read**, **Work Items – Read**).
5. Скопируйте сгенерированный токен — он показывается только один раз.

PAT вводится **не в `.env`**, а через веб-интерфейс приложения на странице **Settings** (`/settings`) после запуска.

### 3. Запустить приложение

**Вариант А — напрямую через Python:**

```bash
pip install -r requirements.txt
python app.py
```

**Вариант Б — через Docker:**

```bash
docker compose up
```

После запуска откройте браузер: [http://localhost:14731](http://localhost:14731)

### 4. Настроить подключение к TFS

1. Перейдите на страницу **Settings** (`/settings`).
2. Укажите TFS URL, Collection и вставьте PAT-токен.
3. Нажмите **Test Connection** — убедитесь, что соединение работает.
4. Сохраните настройки и запустите первую синхронизацию кнопкой **Sync**.
